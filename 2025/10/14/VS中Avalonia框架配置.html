VS中使用Avalonia

Created2025-10-14|Updated2025-10-15|Csharp
|Post Views:

跨平台的“现代WPF”

Avalonia 是一个 跨平台的 .NET UI 框架，专为构建高性能、视觉一致的桌面应用而设计，同时支持扩展到移动和网页平台。

它使用C#编写，依赖.Net SDK编译，既继承了 WPF 等传统 .NET UI 框架的优势，又解决了其跨平台能力的局限性。

在VS中使用

安装VS+SDK

从官网安装VS ,然后在VS Installer里根据开发需求选择工作负载安装。

进行 C# 开发一般需要选择 “.NET 桌面开发” 工作负载，如果要进行ASP.NET和 Web 开发，还需选择 “ASP.NET和 Web 开发” 工作负载。

如果在VS Installer里选了C#相关的工作负载安装，那么.Net SDK就已经包含了，若非，可以自行在官网上找你所需要的SDK安装。
